# Timer Avanzato

Timer Avanzato is a sleek, browser-based advanced timer application built with HTML, CSS, and JavaScript. It offers a user-friendly interface for setting and managing countdowns, with features designed to enhance productivity and time management.

![Timer Avanzato Screenshot](https://via.placeholder.com/600x400.png?text=Timer+Avanzato+Screenshot)

## Features

- **Flexible Time Input**: Set hours, minutes, and seconds with easy-to-use input fields.
- **Dynamic Tab Title**: The browser tab title updates in real-time, showing the current countdown.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.
- **Auto-Repeat Option**: Automatically restart the timer when it reaches zero.
- **Browser Notifications**: Get alerted when your timer ends, even if you're in a different tab.
- **Pause and Reset**: Full control over your timer with pause and reset functionality.

## Usage

1. **Set the Timer**: 
   - Enter the desired hours, minutes, and seconds in the input fields.
   - The main display updates in real-time as you input values.

2. **Start the Timer**: 
   - Click the "Avvia" (Start) button to begin the countdown.
   - The browser tab title will update to show the current time remaining.

3. **Pause and Resume**: 
   - Use the "Pausa" (Pause) button to temporarily stop the timer.
   - Click "Avvia" again to resume from where you left off.

4. **Reset the Timer**: 
   - The "Reset" button clears the current time and stops the timer.

5. **Auto-Repeat**: 
   - Check the "Ripeti automaticamente" box to make the timer restart automatically when it reaches zero.

6. **Notifications**: 
   - Allow browser notifications when prompted to receive alerts when your timer ends.
